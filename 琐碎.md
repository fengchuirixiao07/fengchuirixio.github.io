**1.Quoted-printable编码**
><br>特征：任何一个8位的字节值可编码为3个字符：一个等号”=”后跟随两个十六进制数字(0–9或A–F)表示该字节的数值.
>><br>1.基本编码:任何8位字节（非ASCII字符或特殊字符）可表示为=后跟两个十六进制数字
<br>2.字面表示:十进制值33-60和62-126的可打印ASCII字符通常直接原样输出
<br>3.特殊字符（等号=）​必须被编码为=3D，因为它被用作编码标识符
<br>4.空白字符（空格/Tab）​通常可直接表示，但出现在行尾时，必须编码（空格为=20，Tab为=09）
<br>5.换行符​ 文本中的硬换行需转换为标准CRLF序列（\r\n），在QP编码中即为=0D=0A
>>
```python
import quopri
encoded_data = quopri.encodestring(b"你好，世界！")
decoded_data = quopri.decodestring(b"=E4=BD=A0=E5=A5=BD=EF=BC=8C=E4=B8=96=E7=95=8C=EF=BC=81")
```
